# Contributing to AgenticMinds

Thank you for your interest in contributing to AgenticMinds! We welcome contributions from the community to help make this library better.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or bugfix (`git checkout -b feature/amazing-feature`).
4.  **Make your changes**.
5.  **Run tests** (if applicable) to ensure everything is working.
6.  **Commit your changes** (`git commit -m 'Add some amazing feature'`).
7.  **Push to the branch** (`git push origin feature/amazing-feature`).
8.  **Open a Pull Request**.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the GitHub repository. Provide as much detail as possible to help us understand and resolve the issue.

## Code Style

*   Follow standard Python coding conventions (PEP 8).
*   Keep code clean and readable.
*   Add comments where necessary to explain complex logic.
